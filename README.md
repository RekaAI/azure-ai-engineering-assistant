# Azure AI Engineering Assistant

A secure .NET 8 and Azure OpenAI reference application for searching engineering documentation and generating cited answers.

## Overview

The Azure AI Engineering Assistant is designed to help engineering teams find trusted answers across architecture standards, coding guidelines, runbooks, onboarding documents, and technical documentation.

The project will demonstrate how to build a secure enterprise AI solution using Azure OpenAI, retrieval-augmented generation, and cloud-native engineering practices.

## Planned Capabilities

- Upload and organize engineering documents
- Search technical content using natural language
- Generate answers with source citations
- Support architecture, coding, DevOps, and operational documentation
- Apply secure access and responsible AI practices
- Provide logging, monitoring, and evaluation

## Technology Stack

- .NET 8
- C#
- Azure OpenAI
- Azure AI Search
- Azure Blob Storage
- Angular
- Docker
- Azure DevOps / GitHub Actions

## Project Status

🚧 Planning and architecture phase

## Goals

- Demonstrate enterprise-grade Azure AI architecture
- Explore retrieval-augmented generation
- Apply secure software engineering practices
- Build a practical AI assistant for engineering teams
